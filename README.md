\# 💰 Personal Finance Tracker



A full-stack Personal Finance Tracker web application that helps users manage their income and expenses, track their budget, and monitor their financial health.



\## 🚀 Features



\- 👤 User Registration \& Login

\- 🔐 Secure Authentication

\- 💵 Add Income

\- 💸 Add Expenses

\- 📊 Dashboard with Financial Summary

\- 📅 Transaction History

\- ✏️ Edit \& Delete Transactions

\- 💰 Budget Management

\- 📱 Responsive User Interface



\---



\## 🛠️ Technologies Used



\### Frontend

\- React.js

\- Vite

\- Axios

\- CSS



\### Backend

\- Spring Boot

\- Spring Security

\- Spring Data JPA

\- REST API



\### Database

\- MySQL

\## ⚙️ Installation



\### Clone Repository



```bash

git clone https://github.com/gayatri86pagadala/PersonalFinanceTracker.git

```



\### Backend



```bash

cd backend

```



Configure MySQL in:



```

application.properties

```



Run:



```bash

mvn spring-boot:run

```



\---



\### Frontend



```bash

cd frontend

npm install

npm run dev

```
